\doxysubsection{Detailed Description}
Anything that can be printed with standard output. 

A concept that captures everything that can be printed using the standard output objects, such as {\ttfamily std\+::cout}, {\ttfamily std\+::cin}, or {\ttfamily std\+::clog}. This includes objects who have overrode the {\ttfamily \texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>}} operator for standard ostream.
