\doxysubsection{Detailed Description}
Core Kirho library features. 

In other words, this file contains all of the features that are part of the kirho library. There are, unfortunately, no more files in this library. I suppose that you can say that this is not only a header only library, but also a single-\/header library. Unsure of how long this is going to last, however, as the library could become multifile in the future.

It will create an object named whatever the {\ttfamily name} is, with the term {\ttfamily \+\_\+defer} appended to the end of it. It should explain why you get errors when you try to create objects of such names.
