\subsubsection*{template$<$deferable\+\_\+t F$>$\newline
struct kirho\+::defer\+\_\+t$<$ F $>$}
A way to defer execution of something until the end of the current scope. 

This is essentially a class that provides a way for you to defer execution of something until the end of the current scope. How it works is that you create an object of this type and pass into it\textquotesingle{}s constructor the function or closure that you would like to defer the execution of. Then, the destructor will automatically call the function at the end of the current scope. This process may sound tedious, but there is a macro provided that can a lot of this for you.
